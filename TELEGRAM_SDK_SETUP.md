# Инструкция по подключению Telegram SDK и авторизации пользователей

## Шаг 1: Подключение Telegram SDK

Telegram SDK уже подключен в вашем проекте через CDN в HTML файлах:

```html
<script src="https://telegram.org/js/telegram-web-app.js"></script>
```

**Проверьте, что скрипт подключен во всех HTML файлах!**

## Шаг 2: Создание утилиты для работы с Telegram Auth

Создайте файл `telegram-auth.js` в корне проекта.

## Шаг 3: Интеграция в проект

### 3.1. Добавьте скрипт в HTML файлы

Добавьте перед закрывающим тегом `</body>` в основных HTML файлах:

```html
<script src="telegram-auth.js"></script>
```

### 3.2. Используйте в ваших JS файлах

После инициализации `tg` используйте `TelegramAuth` для работы с пользователем.

## Шаг 4: Настройка бота в Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather):
   ```
   /newbot
   ```

2. Получите токен бота

3. Настройте Web App:
   ```
   /newapp
   ```
   - Выберите вашего бота
   - Укажите название приложения
   - Загрузите иконку (опционально)
   - Укажите URL: `https://your-domain.com` (или локальный для тестирования)

## Шаг 5: Тестирование

### Локальное тестирование:
1. Запустите локальный сервер: `python -m http.server 8000`
2. Используйте ngrok для туннеля: `ngrok http 8000`
3. Укажите ngrok URL в настройках бота

### Продакшн тестирование:
1. Загрузите файлы на сервер
2. Укажите домен в настройках бота
3. Откройте бота в Telegram

## Примеры использования

См. файл `telegram-auth-examples.js` для подробных примеров.
