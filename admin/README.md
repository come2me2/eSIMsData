# Admin Panel - Tailwind CSS Setup

## Описание

Админ-панель использует Tailwind CSS v3 для стилизации. CSS файл предварительно компилируется для production, что значительно ускоряет загрузку страниц.

## Разработка

### При изменении HTML или добавлении новых Tailwind классов:

1. **Автоматическая пересборка во время разработки:**
   ```bash
   npm run watch:css
   ```
   Эта команда будет отслеживать изменения и автоматически перекомпилировать CSS.

2. **Разовая сборка (для production):**
   ```bash
   npm run build:css
   ```

### Файлы

- `admin/css/tailwind-input.css` - входной файл Tailwind (не редактировать)
- `admin/css/tailwind.css` - скомпилированный CSS (автогенерируемый)
- `admin/css/admin.css` - кастомные стили
- `tailwind.config.js` - конфигурация Tailwind

## Важно

❌ **НЕ используйте CDN версию Tailwind CSS** (`cdn.tailwindcss.com`)  
✅ **Всегда используйте предкомпилированный файл** `css/tailwind.css`

CDN версия компилирует стили на лету в браузере, что сильно замедляет работу, особенно на страницах с большим количеством элементов.

## Производительность

- **Размер CSS файла:** ~13KB (минифицированный)
- **Загрузка:** Мгновенная
- **Без runtime компиляции:** Да

## Деплой

Перед деплоем убедитесь, что выполнили:
```bash
npm run build:css
```

Файл `admin/css/tailwind.css` должен быть закоммичен в репозиторий.
